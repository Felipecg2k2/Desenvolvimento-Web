Introdução ao Desenvolvimento Web

O desenvolvimento web é a área da programação responsável pela criação de sites e aplicações que funcionam no navegador (como Chrome, Edge, Firefox, etc.).
Tudo o que vemos na internet — páginas, textos, imagens, botões e layouts — é resultado da combinação de várias tecnologias que trabalham juntas.
Neste repositório, o foco inicial será entender a base do desenvolvimento web, começando pelas tecnologias fundamentais do front-end: HTML e CSS.

O que é um site?
Um site é um conjunto de arquivos que o navegador interpreta e exibe para o usuário.
Esses arquivos geralmente incluem:
HTML → estrutura do conteúdo
CSS → aparência e layout
(Outras tecnologias podem existir, mas não fazem parte deste início)
O navegador lê esses arquivos e transforma o código em uma página visual.

Front-end e Back-end
No desenvolvimento web, é comum dividir o sistema em duas partes:

Front-end
É tudo o que o usuário vê e interage no navegador.
Inclui:
textos
imagens
cores
botões
organização da página

As principais tecnologias do front-end são:
HTML
CSS
JavaScript (não será abordado agora)

Back-end
É a parte que roda no servidor e não aparece diretamente para o usuário.

Geralmente envolve:
lógica de negócio
banco de dados
autenticação
regras do sistema

Neste repositório, não abordaremos back-end por enquanto.

O papel do HTML

O HTML (HyperText Markup Language) é responsável por estruturar o conteúdo da página.
Com HTML, definimos:
títulos
parágrafos
listas
imagens
links
Ele funciona como o esqueleto de um site.

O papel do CSS

O CSS (Cascading Style Sheets) é responsável pela aparência visual da página.

Com CSS, controlamos:
cores
fontes
espaçamentos
alinhamento
layout

Se o HTML é o esqueleto, o CSS é a roupa e o visual do site.

Como o navegador funciona nesse processo?

O navegador:
lê o arquivo HTML
entende a estrutura do conteúdo
aplica os estilos definidos no CSS
exibe o resultado final para o usuário
Todo esse processo acontece automaticamente sempre que uma página é carregada.


Objetivo deste repositório
Este repositório tem como objetivo:
aprender HTML e CSS do zero
entender conceitos básicos antes da prática
manter o conteúdo simples e organizado
evoluir de forma gradual, com commits frequentes

Nada avançado será abordado neste início.
O foco é construir uma base sólida.
